Bonus - Creating a website using Git version control & the unix shell

This page will show an example of creating a web page using Git version control and the unix shell.

Below is a screenshot demonstrating the use commands in the unix shell to manage creating a new local repository and linking it with a remote one:

We can see that we have:

  • Created a new repository called 'command-repo' and a readme file within it.
  • Staged and the commited the changes along with a description of the changes
  • Created a new branch
  • Linked with a remote repository (on GitHub)
  • Cloned the repository
  • Merged the changes into the master branch
